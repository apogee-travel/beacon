# Using Beacon w/React

This is a ridiculously simple example Vite/React app showing the usage of a Beacon store. The store provides a fake product list, and the data is displayed via the ProductList and ProductListItem components. The store also makes use of middleware that loads state from local storage, and persists
updated state to local storage as state changes.

Run `npm install` to install deps.

Run `npm run dev` to start the example app.
