## Core Package

Beacon requires MobX as a peer dependency.

<Tabs>
    <TabItem label="npm">
    ```bash
    npm install @apogeelabs/beacon mobx
    ```
    </TabItem>
    <TabItem label="yarn">
    ```bash
    yarn add @apogeelabs/beacon mobx
    ```
    </TabItem>
    <TabItem label="pnpm">
    ```bash
    pnpm add @apogeelabs/beacon mobx
    ```
    </TabItem>
</Tabs>

## Requirements

- **Node.js** 18+
- **TypeScript** 5.0+ (recommended, not required)
- **MobX** 6.x
- **React** 19+ (for `beacon-react-utils` — uses `useEffectEvent`)

## Package Summary

| Package | Purpose | Required Peers |
|---------|---------|----------------|
| `@apogeelabs/beacon` | Core store, compose, types | `mobx` |
| `@apogeelabs/beacon-browserstorage` | localStorage/sessionStorage persistence | `@apogeelabs/beacon`, `mobx` |
| `@apogeelabs/beacon-react-utils` | `useStoreState`, `useStoreWatcher` hooks | `@apogeelabs/beacon`, `mobx`, `react` |
| `@apogeelabs/beacon-actorstore` | XState actor integration | `@apogeelabs/beacon`, `mobx`, `xstate` |
